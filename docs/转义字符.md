title: 转义字符
---

| 转义序列 | 字符                                       |
| -------- | ------------------------------------------ |
| `\b`     | 退格符                                     |
| `\f`     | 换页符                                     |
| `\n`     | 换行符                                     |
| `\r`     | 回车符                                     |
| `\t`     | 制表符                                     |
| `\unnnn` | Unicode 字符                               |
| `\xnn`   | ASCII 字符，字符代码由十六进制数字 nn 指定 |
| `\'`     | 单引号                                     |
| `\"`     | 双引号                                     |
| `\\`     | 单个反斜杠字符                             |
