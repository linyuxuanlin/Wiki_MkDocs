## 对比

|                          | 27T      | 35T      |
| ------------------------ | -------- | -------- |
| 空载转速（7.4V）（参考） | 16800 转 | 12000 转 |
| 空载转速（12V）          | 29000    | 21000    |
| 空载电流（12V）          | 2.0A     | 1.4A     |
| 发热（10 分钟）          | 52℃      | 46℃      |

匝数越少，转速越快（导致电流越大、发热越大），扭矩越小。
